Square Enix Nier: Automata

Nier: Automata is an action role-playing game developed by PlatinumGames and published by Square Enix and was released in early 2017. This title is set in a post apocalyptic world where machines and remnants of humanity are at war and androids are sent to gather intel and fight off the machines so they can save mankind. This title was nominated for “best storytelling” and “best game of the year” and won the award for “best music”.

For the most part, Nier: Automata is a fast paced hack and slash game that sometimes transitions into a bullet hell style game which is nostalgic to older games like Galaga. Initially the game may seem like something that is as straight forward as “destroy all robots and save the world”, however there is a reason this title was nominated for “best storytelling”. This is a great game for players who enjoy a lot of action and a great storyline.
